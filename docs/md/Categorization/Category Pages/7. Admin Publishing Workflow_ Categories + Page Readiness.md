## **2) NestJS --- Page Health metrics (simple, read-only)**

**Explanation\**
Compute a **readiness** snapshot per category level + page kind
(All/Projects/Services/Teams/Talent) using your **existing rules**. This
is **read-only** and does **not** change your assembler.

**File**: /apps/api/src/category-pages/health.service.ts (new)

import { Injectable } from \'@nestjs/common\'

const MIN_ROWS = Number(process.env.PAGE_HEALTH_MIN_GRID_ROWS \|\| 2)

const MIN_CAROUSEL_PAGES =
Number(process.env.PAGE_HEALTH_MIN_CAROUSEL_PAGES \|\| 3)

\@Injectable()

export class PageHealthService {

async assess({ blocks }: { blocks: Array\<{ type:
\'grid\'\|\'carousel\'; entity: string; items: any\[\]; }\>) {

const checks = blocks.map(b =\> {

if (b.type === \'grid\') {

// Assume gridSize is 3 cards per row unless you attach it in block
spec; adjust if you pass it

const perRow = 3

const minItems = perRow \* MIN_ROWS

return { entity: b.entity, type: b.type, items: b.items.length, pass:
b.items.length \>= minItems, reason: \`\>= \${MIN_ROWS} rows\` }

}

// carousel

const perRow = 3

const minItems = perRow \* MIN_CAROUSEL_PAGES

return { entity: b.entity, type: b.type, items: b.items.length, pass:
b.items.length \>= minItems, reason: \`\>= \${MIN_CAROUSEL_PAGES}
carousel pages\` }

})

const pass = checks.every(c =\> c.pass)

return { pass, checks }

}

}

pageKind: payload.pageKind \|\| \'show-all\' }

## **4) Optional --- Publish gating (soft or hard)**

**Explanation\**
When an editor toggles **Publish** on a category, you can **warn**
(soft) or **block** (hard) if the **Page Health** fails. This doesn't
touch your existing data; it only intercepts the publish action.

**File**: /src/api/category/content-types/category/lifecycles.js\
**Soft gate (warn only):**

****module.exports = {

async beforeUpdate(event) {

const willPublish = !!event.params?.data?.publishedAt &&
!event.params?.data?.unpublishedAt

if (!willPublish) return

try {

// Build the slugPath for this category (L1/L2/L3) using your existing
fields

const slugPath = await
strapi.service(\'api::category.category\').getSlugPath(event.params.where.id)

const url =
\`\${process.env.NEXT_PUBLIC_API_URL}/internal/page-health/\${slugPath.join(\'/\')}\`

const res = await fetch(\`\${url}?preview=1\`, { headers: {
\'x-preview-token\': process.env.PREVIEW_TOKEN } })

const health = await res.json()

if (!health.pass) {

// Soft gate: add a warning field (no throw)

event.params.data.pageHealthWarning = \'Publishing while not ready (see
Page Health meter).\'

}

} catch (e) { /\* fail open \*/ }

}

}

}

- Unpublished L2/L3 preview renders (noindex).

  purge (no changes needed).
