# **Backend -- Auth & Access Control**

Purpose: Secure the Portfolio Service by adding JWT-based authentication
and access guards. Ensure only portfolio owners or admins can access
sensitive endpoints (/draft, /publish, /preview). Configure CORS to
allow cross-service requests safely.

## **0) Prerequisites**

- Repo path: apps/portfolio-service

- Auth folder: src/auth/

- Existing API gateway or auth service issues JWTs (with claims: sub =
  user ID, role = user role)

- Environment variables:

  - JWT_PUBLIC_KEY (PEM-encoded RSA public key if RS256)

  - or JWT_SECRET (if HS256 symmetric)

  - ALLOWED_ORIGINS (comma-separated list of frontend domains)

## **3) JWT Strategy**

> File: src/auth/jwt.strategy.ts

import { Injectable } from \"@nestjs/common\";

import { PassportStrategy } from \"@nestjs/passport\";

import { ExtractJwt, Strategy } from \"passport-jwt\";

\@Injectable()

export class JwtStrategy extends PassportStrategy(Strategy) {

constructor() {

super({

jwtFromRequest: ExtractJwt.fromAuthHeaderAsBearerToken(),

secretOrKey: process.env.JWT_SECRET \|\| process.env.JWT_PUBLIC_KEY,

algorithms: \[process.env.JWT_PUBLIC_KEY ? \"RS256\" : \"HS256\"\],

});

}

async validate(payload: any) {

return { userId: payload.sub, role: payload.role };

}

}

This makes decoded JWT payload available as req.user.

## **4) Roles Guard**

> File: src/auth/roles.guard.ts

import { Injectable, CanActivate, ExecutionContext, ForbiddenException
} from \"@nestjs/common\";

import { Reflector } from \"@nestjs/core\";

\@Injectable()

export class RolesGuard implements CanActivate {

constructor(private reflector: Reflector) {}

canActivate(context: ExecutionContext): boolean {

const request = context.switchToHttp().getRequest();

const user = request.user;

if (!user) throw new ForbiddenException(\"No user context\");

// Only allow admins or owner for protected routes

const params = request.params \|\| {};

const body = request.body \|\| {};

const ownerId = body.ownerUserId \|\| request.query.owner;

if (user.role === \"admin\") return true;

if (ownerId && user.userId === ownerId) return true;

throw new ForbiddenException(\"Access denied\");

}

}

## **8) Best Practices**

- Never expose /draft or /preview publicly.

- Keep ALLOWED_ORIGINS strict (only your Next.js domains).

- Rotate JWT secrets/keys periodically.

- Log failed auth attempts for auditing.

This document secures the Portfolio Service with **JWT-based
authentication**, **role/ownership guards**, and **CORS configuration**.
It ensures that only authorized users can modify or preview portfolio
items.
