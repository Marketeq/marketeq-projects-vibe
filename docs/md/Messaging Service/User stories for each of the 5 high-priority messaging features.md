**User stories** for each of the 5 high-priority messaging features.

These are written in the format:

> **As a \[user type\], I want to \[do something\] so that
> \[benefit/result\].**

## **🧠 Typing Indicators**

> **As a user**, I want to see when the other person is typing in a
> conversation so that I know they're actively replying and I don't
> interrupt them or send unnecessary follow-ups.

## **👁️‍🗨️ Read Receipts**

> **As a user**, I want to know when my messages have been read by the
> recipient so that I can feel confident my communication was received
> and avoid repeating myself.
>
> **As a client**, I want to know if a talent has read my message so I
> can manage my follow-ups without being too pushy or assuming they\'re
> ignoring me.

## **😀 Message Reactions**

> **As a user**, I want to react to a message with an emoji so that I
> can quickly respond or show appreciation without sending another
> text-based reply.
>
> **As a client**, I want to use reactions to acknowledge progress
> updates or confirm simple decisions without cluttering the chat
> thread.

## **🚫 Block / Report Controls**

> **As a user**, I want the ability to block someone from messaging me
> so that I feel safe and in control if someone behaves inappropriately
> or unprofessionally.
>
> **As a talent**, I want to report a user if they attempt to solicit me
> off-platform or send spam so that the platform can take appropriate
> action to protect others.

## **🗂️ Inbox Filtering Tabs**

> **As a user**, I want to easily filter my inbox by categories like
> unread, starred, or archived so that I can find and prioritize
> important conversations quickly.
>
> **As a power user or agency**, I want to manage a large number of
> threads efficiently using inbox filters so that I don't miss urgent
> updates or repeat communication.
