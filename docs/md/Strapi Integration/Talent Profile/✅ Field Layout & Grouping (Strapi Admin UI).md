## **✅ Field Layout & Grouping (Strapi Admin UI)**

### **🔝 Top Section (Basic Identity)**

- **Username** (left column)

- **Availability** (right column as a dropdown)

- 📌 *Placement: top row\*

### **📍 Location & Time Zone (1 row, 2 columns)**

- **Location** (left)

- **Time Zone** (right)\
  → **Best practice:** Use a **dropdown with autocomplete** (e.g.
  select2 behavior) with \~500+ standard time zones like
  America/New_York, Asia/Kolkata.

### **🌐 Skills**

- **Skills** (full-width row)\
  → Autocomplete + tag entry, pre-validated from central skills table

### **📝 Overview**

- **Bio / Overview** (full-width row)\
  → Rich text editor

### **🗂 Certifications, Industry, Languages**

- **Certifications** (left)

- **Industry Expertise** (right)

- **Languages** (full-width row below)

### **🎓 Education (own row, no section title)**

- **Education** (full-width)

  - Repeatable component: Institution, Degree, Graduation Year

### **💼 Work Experience (own row, no section title)**

- **Work Experience** (full-width)

  - Repeatable component: Role, Company, Start/End Date, Description
